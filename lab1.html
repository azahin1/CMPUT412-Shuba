Basic Development

For development, of course Git and Github is required. To develop docker images and uploading them to dockerhub with python, a place to start is needed. Thankfully duckiebots are open source and available in Github. Forking that repository and using it as a base to run code from the bot was a perfect start.

However, problems arose imediately when running Hello World through the bot. As seen below in the images, I could develop the package, create an image locally, and run it. But as soon as I tried to create the image from the bot using Docker, errors constantly popped up.

To get to the root of the problem, I SSHed into the duckiebot and tried to log in to Docker. It failed. So I pinged Google to see if it was connected to the network. It was. So the problem was Docker. So I uninstalled and re-installed Docker, and it worked back up again. At the end, the image was made, and able to work properly.

Then I tried to do some colour detection with the camera, but the credentials in the examples seemed to be off and it didn't work with anything. At this point, I ran out of time for the exercise so this part I couldn't finish.
